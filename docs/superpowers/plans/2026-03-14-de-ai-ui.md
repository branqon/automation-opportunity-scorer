# De-AI the UI — Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Remove AI-generated dashboard aesthetic by establishing visual hierarchy, replacing purple accent with slate, removing all border radius, and restructuring the dashboard so the ranked table is the primary content.

**Architecture:** Token-level color and radius changes in globals.css propagate to all components via CSS variables. Layout changes concentrate in dashboard-client.tsx (remove summary cards, remove top candidates, reorder content, collapsible charts). Detail page gets surface flattening. E2E tests updated for new structure.

**Tech Stack:** Next.js 16, React 19, Tailwind CSS 4, Recharts, Playwright

**Spec:** `docs/superpowers/specs/2026-03-14-de-ai-ui-design.md`
