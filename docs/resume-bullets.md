# Resume Bullet Ideas

- Built `automation-opportunity-scorer`, a Next.js and Prisma internal-tool style application that ranks recurring operational work by business value, implementation fit, and visible ROI assumptions.
- Designed a deterministic automation-prioritization model that combined workload volume, analyst effort, repeatability, SLA risk, customer impact, and delivery complexity into a transparent 100-point score, plus a what-if weighting mode for scenario analysis.
- Modeled a realistic service-operations dataset in SQLite, exported the app as a static site, and deployed it through GitHub Pages with no runtime server dependency.
- Created opportunity detail views that translated scoring outputs into concrete implementation next steps such as self-service flows, API provisioning, scheduled follow-up automation, and AI-assisted triage.
- Added Playwright end-to-end coverage for dashboard filtering, weight-driven reranking, and detail-page navigation alongside unit tests for scoring and URL-state helpers.
- Delivered a portfolio-ready analytics product with production-style TypeScript structure, responsive internal-tool UI, seeded setup, CI verification, and documentation covering scoring methodology and engineering tradeoffs.
