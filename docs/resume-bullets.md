# Resume Bullet Ideas

- Built `automation-opportunity-scorer`, a Next.js and Prisma internal-tool style application that ranks recurring operational work by business value, implementation fit, and visible ROI assumptions.
- Designed a deterministic automation-prioritization model that combined workload volume, analyst effort, repeatability, SLA risk, customer impact, and delivery complexity into a transparent 100-point score.
- Modeled a realistic service-operations dataset in SQLite and shipped a decision-support dashboard with ranked opportunities, savings estimates, and quick-win versus strategic-bet segmentation.
- Created opportunity detail views that translated scoring outputs into concrete implementation next steps such as self-service flows, API provisioning, scheduled follow-up automation, and AI-assisted triage.
- Delivered a portfolio-ready analytics product with production-style TypeScript structure, responsive internal-tool UI, seeded local setup, and documentation covering scoring methodology and engineering tradeoffs.
